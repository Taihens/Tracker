Pour synchroniser les données entre appareils en temps réel, créez un projet gratuit sur firebase.google.com et collez votre config ci-dessous.

Étapes :
1. firebase.google.com → Nouveau projet
2. Realtime Database → Créer → Mode test
3. Paramètres du projet → Config web → Copier
4. Collez la config dans le fichier HTML à l'endroit indiqué
// Remplacer dans le JS :
const FIREBASE_CONFIG = {
  apiKey: "VOTRE_CLE",
  databaseURL: "VOTRE_URL",
  projectId: "VOTRE_ID"
};
